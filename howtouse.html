📖 How to Use This Site

A step-by-step guide to using the Innovation Club Voting System.

Getting Started

  1. Open the site: Use the official link shared by Innovation Club DPS Newtown.
  2. Check status: If the landing page shows voting or member login as unavailable, try again later.

Voting

  1. Open the Voting page when it’s active.
  2. Read the question and available options.
  3. Select your choice and submit. If voting is closed, the page will clearly say so.

Tips

  • Use the latest version of Chrome or Edge.
  • Check your network connection if pages don’t update.
  • For issues, contact the Innovation Club team and include your Admission Number.

FAQ

My admission number isn't found

Please verify the format (letters + numbers, if applicable). If still not found, contact the admins to check the membership list.

Voting says it is closed

When the voting session is inactive, the page will display a closed message. Try again during the announced time window.

Attendance looks incorrect

Use the edit request feature from your profile. Provide the old and corrected value; admins will review it.

🚀 Release Notes v2.3.0

Hey Innovators! 🌟 v2.3.0 is here with solid upgrades and fixes. This update focuses on making the site more powerful, admin-friendly, and reliable. Here’s the breakdown:

  • 🔐 Admin Page Upgrade: Admins now get easier, central access to all key features. ⚡
  • 📢 Dynamic Announcements: Announcements are now pulled live from a Google Sheets file instead of hardcoded JSON. 📄
  • 🎨 Button UI Improvement: Sleeker buttons added to all themes for a consistent, modern feel. 🎀
  • 🗳️ Voters Page: Improved design & functionality — the voters page is now fully open and ready. ✅
  • 🌐 Webhook Fix: Added proper CORS headers to prevent those annoying “Network Error” messages. 🛡️
  • ⌨️ Typing Effect: Added a subtle animated typing effect for headers and key text to increase visual engagement. ✨
  • ⚙️ Full Functionality: All major modules are now working together seamlessly for smoother operation. 🔄
  • 🧑‍💻 Upcoming: Member page is still being built — expect it in the next version. 👷‍♀️
  • 🐞 Bug Fixes: Squashed various bugs to improve overall stability and performance. 🐛


These notes cover update 2.3.0. Static & final. The next release notes will drop with version 2.4.0.
